# Netflix Data Analysis - Task 3

## Problem Statement
The goal of this project is to perform exploratory data analysis (EDA) on the Netflix dataset to identify trends, patterns, and insights from the data.

## Dataset Description
The dataset contains information about Netflix movies and TV shows, including:
- Title details
- Type (Movie or TV Show)
- Director and cast
- Country of production
- Release year
- Rating
- Duration
- Genre

## Approach
The project followed a structured exploratory data analysis process:

1. Loaded the dataset using Pandas
2. Performed data inspection and summary statistics
3. Checked and handled missing values
4. Removed duplicate records
5. Converted date columns to proper datetime format
6. Performed correlation analysis on numeric features
7. Conducted trend analysis:
   - Movies vs TV Shows distribution
   - Content release trends over years
   - Top countries producing content
   - Ratings distribution

## Results
After analysis, the following insights were found:
- Netflix has more movies than TV shows
- Content production increased significantly after 2010
- The United States is the largest content producer
- TV-MA is the most common rating on Netflix

The dataset is now well-understood and ready for further analysis or modeling.
